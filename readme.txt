This is a project developed for the Web Mobile Development classes at Faculdade Senac Porto Alegre.

Project goals:
• Grid must support an 8 column layout;
• Gutters must have 1.5em;
• Grid must show only one column in mobile and tablet (1024px or less);
• There shouldn't be gutters in mobile;
• User must develop an website with the minimum of 3 combinations of columns.
	E.g.: (8; 4 + 4; 5 + 3) or (7 + 1; 6 + 2; 3 + 5).
